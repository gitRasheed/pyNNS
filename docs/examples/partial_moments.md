# Partial Moments

This example mirrors the core idea from the R partial-moments vignette: variance
can be decomposed into lower and upper partial moments around the mean.

Run:

```bash
uv run python docs/examples/partial_moments.py
```

The script demonstrates:

- `lpm` and `upm`
- degree-zero probability split
- variance decomposition around the mean
- `nns_moments`
